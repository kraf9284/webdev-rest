Tools Used

To create this application, we used the vue framework and the leaflet api. Vue alluwed us to easily interweave our html elements and javascript code. We connected to the leaflet api, which is a free map service similar to google maps that allowed us to query for locations on the globe. Leaflet allowed us to drop markers on the map so we could represent locations on the map, and with the help of our database, we could easily find locations in the database and plot them on the map.

Interesting Findings

  1. Leaflets API can be buggy and allow for map markers to not stick to the map. Things like google maps allow for a more premium version of this same software, but it costs money
  2. Once you have an API created like we did for our database, it becomes much easier to execute sql queries
  3. Learning API syntax is difficult, online resources really help!
  4. Working with github, it is important to stash your changes so you can pull other changes, then merge the two
  5. Vue allows for much easier implementation of html and javascript
  6. Doing research to figure out how to do something easier, is always worth it, even if it takes more time
